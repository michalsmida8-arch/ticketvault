TicketVault — Přidání macOS buildů
===================================

Tento ZIP přidá Mac build do tvého GitHub Actions workflow.
Po nasazení každý nový tag (v1.6.2, v1.7.0, ...) zbuildí AUTOMATICKY:

  • Windows installer (.exe)
  • Windows portable (.exe)
  • Mac DMG pro Apple Silicon (M1/M2/M3/M4) — arm64
  • Mac DMG pro Intel Macy — x64
  • Mac ZIPy (pro auto-update)

INSTALACE — JEDNOU
===================

Z Windows PC:

  cd /c/Users/micha/Desktop/ticketvault
  
  # 1. Rozbal tento ZIP do své ticketvault složky
  #    (přepíše .github/workflows/release.yml)
  
  # 2. Push na GitHub
  git add .github/workflows/release.yml
  git commit -m "ci: add macOS to release workflow"
  git push

POUŽITÍ
=======

Příští release už bude buildit Win + Mac:

  git tag v1.6.2
  git push --tags

Sleduj https://github.com/michalsmida8-arch/TicketVault/actions
Trvání: ~10-15 minut (Mac buildy jsou pomalejší)

V Releases pak budou tyto soubory:
  • TicketVault-Setup-1.6.2.exe          (Windows installer)
  • TicketVault-Portable-1.6.2.exe       (Windows portable)
  • TicketVault-1.6.2-arm64.dmg          (Mac M1/M2/M3/M4)
  • TicketVault-1.6.2-x64.dmg            (Mac Intel)
  • TicketVault-1.6.2-arm64-mac.zip      (Mac auto-update arm64)
  • TicketVault-1.6.2-x64-mac.zip        (Mac auto-update Intel)
  • latest.yml                            (Win auto-update manifest)
  • latest-mac.yml                        (Mac auto-update manifest)

INSTALACE NA MAC (PRVNÍ INSTALACE)
===================================

1. Stáhni .dmg pro svůj Mac (arm64 = M1+, x64 = Intel)
2. Otevři .dmg → přetáhni TicketVault do Applications

Při PRVNÍM spuštění macOS zablokuje appku ("nelze ověřit developera"),
protože nemáme Apple Developer účet ($99/rok).

Řešení (jen poprvé):
  • PRAVÝ KLIK na TicketVault.app → Otevřít → Otevřít

Nebo v terminálu:
  xattr -cr /Applications/TicketVault.app

macOS si to zapamatuje, příště startuje normálně.

AUTO-UPDATE NA MAC
==================
Funguje stejně jako na Windows — appka si při dalším spuštění
sama stáhne novou verzi z GitHub Releases.
